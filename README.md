# Сериализация двусвязного списка со случайными указателями

## Описание
Проект реализует **сериализацию и десериализацию двусвязного списка со случайными указателями** (C++17).  
Каждый узел содержит:
- `prev` — указатель на предыдущий узел;
- `next` — указатель на следующий узел;
- `rand` — указатель на произвольный узел того же списка или `nullptr`;
- `data` — строковые пользовательские данные (UTF-8).

Программа:
1. Считывает текстовое описание списка из входного файла.
2. Строит список в памяти.
3. Сериализует список в бинарный файл.
4. Десериализует и проверяет корректность (round-trip).
5. Повторяет операцию по желанию пользователя без перезапуска.

Реализовано с прицелом на большие данные — до **1 000 000** узлов.

---

## Структура узла

```cpp
struct ListNode {
    ListNode* prev = nullptr;
    ListNode* next = nullptr;
    ListNode* rand = nullptr;
    std::string data;
};
```
# Сборка

Требуется компилятор с поддержкой C++17.

# Visual Studio

Необходимо открыть .sln, установить стандарт C++17 (Project → Properties → C/C++ → Language → C++ Language Standard → /std:c++17), собрать проект.

## Запуск

Запуск программы без аргументов, чтобы получить интерактивный ввод

Программа спросит путь к входному файлу.
Если нажать Enter — будет использован inlet.in в текущей рабочей папке.

Выходной файл по умолчанию будет создан рядом с входным файлом как: outlet.out

# Указать входной файл
./list_serializer path/to/inlet.in

# Указать входной и выходной файл
./list_serializer path/to/inlet.in path/to/outlet.out
